# 🏦 Bank Customer Analytics

## 📌 Project Overview

This project analyzes a retail bank's customer data to identify high-value customers, evaluate loan risk, understand customer segments, and support business decision-making using **SQL, Python, and Power BI**.

## 🎯 Business Problem

The bank lacked a structured, data-driven approach to analyze customer behavior, identify profitable customer segments, monitor loan risk, and improve relationship management.

## 🛠️ Tech Stack

* SQL (MySQL)
* Python (Pandas, NumPy, Matplotlib)
* Power BI

## 📊 Key Analysis

* Top Customers by Bank Deposits
* High-Income, Low-Deposit Customer Identification
* Occupation-wise Deposit Analysis
* Loan-to-Income Risk Analysis
* Loyalty Segment Analysis
* Nationality-wise Deposit Analysis
* Interactive Power BI Dashboard

## 📈 Dashboard Features

* KPI Cards (Customers, Deposits, Loans, Income)
* Occupation-wise Deposits & Loans
* Loyalty Distribution
* Nationality Analysis
* Interactive Slicers

## 💡 Business Impact

* Identified high-value customers for relationship management.
* Highlighted customer segments with potential loan repayment risk.
* Supported customer segmentation and targeted marketing strategies.
* Enabled faster, data-driven business decisions through an interactive dashboard.
